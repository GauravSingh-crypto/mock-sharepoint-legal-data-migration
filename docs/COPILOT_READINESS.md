# Copilot readiness considerations

The project prepares documents for discoverability by creating stable business identifiers, controlled metadata, shallow paths, traceability, and explicit exception handling.

A production readiness review must additionally verify:

- Least-privilege SharePoint permissions and oversharing risk
- Sensitivity labels and retention requirements for legal records
- Search indexing status and excluded content
- Duplicate, obsolete, and trivial content handling
- Content-type and required-column completeness
- Representative Microsoft Search and Copilot queries for each security role

Copilot follows the permissions and searchable content available to the requesting user. This project does not bypass or replace Microsoft 365 governance.
